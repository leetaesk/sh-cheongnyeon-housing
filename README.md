# SH 청년 매입임대주택 뷰어

**👉 바로 보기: https://sh-cheongnyeon-housing.vercel.app/**

서울주택도시공사(SH)에서 공고한 **2026년 1차 청년 매입임대주택** 모집 정보를 편하게 둘러보려고 만든 웹 페이지입니다. 공고 원본 자료를 모아서 **주택 리스트 · 위치(지도) · 가격 · 사진**을 한 화면에서 볼 수 있게 정리했습니다. 로컬에서는 `index.html`을 브라우저로 열면 동일하게 동작합니다.

## 무엇이 담겨 있나요?

- **147개 건물 / 849호** (서울 23개 자치구, 신규공급 · 재공급)
- 건물별 **위치 지도**와 네이버 지도 바로가기
- 순위·계층별 **보증금 / 월 임대료** 가격표
- 건물 **사진 갤러리**와 **도면 PDF**

> 참고용으로 만든 개인 정리본이며, 정확한 신청 자격·일정·조건은 반드시
> [SH 청년안심주택 공식 홈페이지](https://soco.seoul.go.kr/youth/main/main.do)와
> [공고문](https://soco.seoul.go.kr/youth/bbs/BMSR00015/view.do?boardId=6485&menuNo=400008)에서 확인하세요.
